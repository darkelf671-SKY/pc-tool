실행하면 내 파일이 삭제되나요?
아니요. 모든 도구는 시스템 캐시나 임시 파일만 정리합니다. 문서, 사진, 작업 파일은 절대 삭제하지 않습니다.
재부팅해야 하나요?
대부분 재부팅 없이 바로 적용됩니다. 필요한 경우 태그에 "재부팅 권장"이라고 표시되어 있습니다.
실행했는데 안 되면 어떻게 하나요?
완료 화면의 안내를 따라해보세요. 그래도 안 되면 상단 "원격 지원 요청"을 클릭해 전산팀에 도움을 요청하세요.
같은 도구를 여러 번 실행해도 되나요?
네, 모든 도구는 여러 번 실행해도 안전합니다. 이미 정상인 항목은 건너뛰고 문제가 있는 부분만 처리합니다.
업데이트는 어떻게 하나요?
프로그램 시작 시 자동으로 최신 버전을 확인합니다. 새 버전이 있으면 알림이 뜨고, 클릭하면 자동 업데이트됩니다.
